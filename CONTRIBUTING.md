# Contributing to Awesome GitHub Profile README

Thank you for your interest in contributing! 🎉

## How to Add a Profile

1. **Fork** this repository
2. Find the appropriate **category** for the profile
3. Add an entry in the format:
   ```
   | - | ![avatar](https://github.com/USERNAME.png?size=40) | [Display Name](https://github.com/USERNAME) | ⭐ Xk | 👥 X | [Profile](https://github.com/USERNAME) | [README](https://github.com/USERNAME/USERNAME) |
   ```
4. The rank/stars will be **automatically updated** periodically
5. Submit a **Pull Request** with a brief description

## Guidelines

- The profile must have a creative or well-designed `README.md` in their profile repository
- No spam or low-effort profiles
- Respect the existing category structure
- One PR per profile addition

## Categories

- **GitHub Actions 🤖** — Profiles powered by GitHub Actions automation
- **Dynamic Realtime 💫** — Live-updating stats and widgets
- **Game Mode 🚀** — Interactive games in profile READMEs  
- **Code Mode 👨🏽‍💻** — Code-focused creative profiles
- **A Little Bit of Everything 😃** — Mixed creative elements
- **Descriptive 🗒** — Well-written, informative profiles
- **Simple but Innovative 🤗** — Clean but cleverly creative
- **Typing.. Mode 🎰** — Animated typing effects
- **Minimalistic ✨** — Clean, minimal aesthetic
- **GIFs 👻** — Profile featuring cool animations/GIFs
- **Anime 👾** — Anime-themed profiles
- **Just Images 🎭** — Image-focused profiles
- **Badges 🎫** — Badge-heavy profiles
- **Icons 🎯** — Icon-focused profiles
- **Fancy Fonts 🖋** — Creative typography
- **Retro 😎** — Retro/pixel art aesthetics
